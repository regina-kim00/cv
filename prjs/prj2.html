Bumblebee Lyon: Memes recommendation system based on the context

#NLP

#Meme

#Recommendation System

Riccardo Tommasini(INSA Lyon) advised this project.

[ summary ] Bumblebee is a system that recommends memes that can be applied for online one-to-one conversations. We first selected 30 memes and labeled each meme with several categories(e.g. ‘happy’, ‘stressful’) before building a whole program. Response Generator gets an input text from the user and generates an output which is the response of the input text. Next, Label Classifier gets the previous output as an input and classifies it into the labels so that it produces the output which is five labels most related to the response. Meme Selector gets these five labels as input and ranks the meme templates based on their relevance. Then the five top templates are chosen as an output.
